# Accelerating the Lebwohl-Lasher Model in python

SCIFM0004 2026

This mini project aimed to implement various acceleration techniques
in order to optimise a Lebwohl-Lasher model written in Python.

Details of usage are within the docstrings of each version, as well as 
extra embedded README files within version directories. 

The Jupyter Notebook contains performance, scaling and validation plots to compare
all the acceleration methods attempted.
